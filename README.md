A simple social-style web application built with Node.js, Express, MongoDB, and EJS.
It supports user authentication, profile pages, posting content, and image uploads.

Features

User registration & login (JWT + cookies)

Password hashing with bcrypt

Protected routes using middleware

User profiles with posts

Create posts linked to users

Image upload using Multer

EJS templating engine

Tech Stack

Node.js

Express.js

MongoDB (Mongoose)

EJS

JWT Authentication

Multer (file uploads)

Bcrypt (password hashing)
